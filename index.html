Introduction


This website is used to provide me overview of FOX Fload DPS 2022 rear shock tunability.
All links available can be used to find what is needed to find tune kit to change settings of damper from Rx Tune to linear.
Rx Tune is default tune provided by Specialized. It has digressive settings which brins very stiff rear at low speeds.
Linear compression tune shoud behave more linear which bring consistent damping during the whole stroke.
Progression is gained from Air chamber volume spacer tune which is by default set by 0.6inch token.
